# Getting Started
## Backend setup

### Symfony
#### Migration and fixtures


### - Go to the backend directory
```bash
cd backend
```

### 1- Install dependencies
```bash
composer install
```

2- Create database
```bash
php bin/console d:d:c
```

3- Run migrations
```bash
php bin/console d:m:m
```

4- Load fixtures
```bash
php bin/console d:f:l
```
